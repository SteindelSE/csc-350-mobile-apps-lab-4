How To Use This App

Select the Number of Passwords

On the first slider, labelled "Number of Passwords", drag the slider or enter the value to indicate the number of random passwords you would like to generate. This can be between 1 and 100 passwords at at time.

Select the Number of Characters

On the second slider, labelled "Number of Characters", drag the slider or enter the value to indicate the number of characters you would like each generated password to have. This can be between 8 and 100 characters
